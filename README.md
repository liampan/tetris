# Tetr-ish

A terminal based tetris-like game.

Clone : `git@github.com:liampan/tetris.git`

in a full screen terminal 

Run : `sbt run`

controls are AWSD or arrow keys.  
  A  < move left  
  D  > move Right  
  W  ^ Roate
